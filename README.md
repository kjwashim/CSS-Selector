# Laundry Mart

A simple HTML and CSS project for a laundry service webpage.

---

## How to Use

* Open the website in your browser
* View available services and pricing
* Fill the form (for demo purpose only)

---

## How to Run

1. Download or clone the project:

   ```bash
   git clone https://github.com/kjwashim/CSS-Selector.git
   ```

2. Open the project folder:

   ```bash
   cd CSS-Selector
   ```

3. Run the project:

   * Double click `index.html`
     OR
   * Right click and open with any web browser

* HTML
* CSS
